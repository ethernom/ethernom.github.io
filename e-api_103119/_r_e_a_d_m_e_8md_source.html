90 ### How-To: Write to devices:
91 Parameter: Inbound Message Struct, Outbound Message Struct & Callback function<br />
92 Callback: resultCode and Inbound Message
93 ```bash
94 //Options for Outbound message variable types are: ETH_STRING_AS_UNICODE, ETH_STRING_AS_ASUTF8, ETH_INT_AS_BYTE & ETH_INT_AS_2BYTE
95 //Outbound message is what will be send to the car
96 //Each parameter must contains the value
97 var out_msg = {
98  command: {
99  encoding: ETH_INT_AS_BYTE,
100  value: TRK_H2C_INIT
101  },
102  PARAMETER1: {
103  encoding: ETH_STRING_AS_ASTUF8,
104  value: "STRING_VALUE";
105  },
106  PARAMETER2: {
107  encoding: ETH_STRING_AS_UNICODE,
108  value: "汉字";
109  },
110  PARAMETER3: {
111  encoding: ETH_INT_AS_2BYTE,
112  value: integer (0 ~ 65535);
113  },
114  PARAMETER4: {
115  encoding: ETH_INT_AS_BYTE,
116  value: integer (0 ~ 255);
117  }
118 }
119 
120 //Options for Inbound message variable types are: ETH_UNICODESTRING, ETH_UTF8STRING, ETH_BYTE, ETH_2BYTE, ETH_PAYLOAD
121 //Inbound message is to define to what's the expected value back from the card, when the EthernomAPI receives the message from card.
122 //The EthernomAPI will know which value belongs to which parameter
123 var in_msg = {
124  command : ETH_BYTE,
125  PARAMETER1: ETH_BYTE,
126  PARAMETER2: ETH_2BYTE,
127  PARAMETER3: ETH_UTF8STRING,
128  PARAMETER4: ETH_UNICODESTRING,
129  PAYLOAD: ETH_PAYLOAD (Optional)
130 }
131 
132 E_API.WriteJSON(out_msg, in_msg, (resultCode, msg) => {
133  if(resultCode == ETH_SUCCESS){
134  console.log(msg.command):
135  console.log(msg.PARAMETER1):
136  console.log(msg.PARAMETER2):
137  console.log(msg.PARAMETER3):
138  console.log(msg.PARAMETER4):
139  console.log(msg.PAYLOAD):
140  }else{
141  //Fails write
142  }
143 }
144 ```
